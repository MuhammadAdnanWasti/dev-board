Question-1: What are the different ways to select an element in the DOM?

There are many different ways like by selecting getElementById, getElementsByClassName, getElementsByName, getElementsByTagName, querySelector, querySelectorAll etc.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML is use to get or set inside Html, innerText just to get or set inside text ,textContent to get or set all text, including hidden elements.

Question-3: What is event delegation in the DOM?

Event delegation helps us to efficiently handle events on multiple child by attaching a single event listener to parent element.

Question-4: What is event bubbling in the DOM?

When a event occurs in child element it's bubble up to it ancestors
